# CSS Folder Structure

This folder contains the CSS files for different pages of the Puzzle of Blessings website.

## Current Files

- **landingpage.css** - Stylesheet for the landing page (index.php)

## Future CSS Files

As new pages are added to the website, create corresponding CSS files here following the naming convention:

- `pagename.css` - Where `pagename` matches the PHP file name

## Example Structure

```
css/
├── landingpage.css    (for index.php)
├── reports.css        (for reports.php - future)
├── profile.css        (for profile.php - future)
├── login.css          (for login.php - future)
└── ...
```

## Notes

- All CSS files should follow the low-poly aesthetic with the color palette defined in `landingpage.css`
- Consider creating a `shared.css` or `variables.css` file in the future for common styles and CSS variables
